# Uinput

Structs and Constants of input, ioctrl and uinput to use uinput with ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'uinput'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uinput

## (Re-)Compiling the binding

```
$ rake ffi:generate
```
